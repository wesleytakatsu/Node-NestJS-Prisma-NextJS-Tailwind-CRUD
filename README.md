## Tecnologias usadas:
Servidores em NodeJS  
  
Backend:
- NestJS Framework
- Prisma ORM
- Rotas APIRest com class-validator
- SQLite
- JWT (em implementação)
  
Frontend:
- NextJS Framework
- Tailwind CSS
- Typescript
- LocalStorage
- Seção com JWT local (em implementação)


## Observações do projeto

- Ao retornar informações do usuário é suprimido o password
